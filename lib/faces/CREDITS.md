# Face art credits

Character sprites in this directory are sourced from
[pokemon-colorscripts](https://gitlab.com/phoneybadger/pokemon-colorscripts)
by phoneybadger, licensed under MIT.

Each preset IS a pokemon — the name in the UI matches the file:

- chansey.txt
- pikachu.txt
- alakazam.txt
- meowth.txt
- charmander.txt
- machamp.txt
- gyarados.txt
- gengar.txt

`custom.txt` is a fallback ASCII emoticon, unused in normal operation (label
logic always resolves to a pokemon name via intensity fallback).

Pokemon and Pokemon character names are trademarks of Nintendo/Game Freak.
The sprite art is distributed via pokemon-colorscripts under MIT.
